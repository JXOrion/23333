HTML

Язык разметки гипертекста (англ. HyperText Markup Language, называемый HTML) — это стандартный язык разметки, используемый для создания веб-страниц.

CSS

Каскадные таблицы стилей (англ. Cascading Style Sheets) — это компьютерный язык, используемый для добавления стилей (шрифтов, интервалов, цветов и т. д.) к структурированным документам (таким как документы HTML или HTML), определяемый и поддерживаемый W3C. CSS3 сейчас поддерживается большинством современных браузеров, а следующая версия CSS4 все еще находится в стадии разработки.

BORDER

Свойство CSS border — это сокращенное свойство для установки различных индивидуальных свойств границы. border можно использовать для установки значения одного или нескольких из следующих свойств: border-width, border-style, border-color.

MARGIN

margin, который является синтаксисом CSS, является сокращенным свойством, которое используется для задания ширины всех полей всех текущих или указанных элементов в объявлении или ширины полей на каждом ребре.

ТАБЛИЧНАЯ ВЕРСТКА

Решетка — это структурированный набор данных (табличные данные), состоящий из строк и столбцов, а общая фраза упорядочивает данные и изображения, что является распространенным элементом веб-разработки

СЕТКА

Сетка — это набор пересекающихся горизонтальных и вертикальных линий, определяющих столбцы и строки сетки.CSS предоставляет систему макета на основе сетки со строками и столбцами, которая упрощает нам разработку веб-страниц без необходимости использования плавающих элементов и позиционирования.

BOOTSTRAP

Bootstrap — это набор фронтенд-фреймворков с открытым исходным кодом для разработки веб-сайтов и веб-приложений, включая HTML, CSS-фреймворки, предоставляющие расширения для шрифтовой типографики, форм, кнопок, навигации и различных других компонентов, предназначенных для упрощения разработки динамических веб-страниц и веб-приложений.

БЛОЧНАЯ ВЕРСТКА

Теги div можно использовать для разделения структуры HTML и, таким образом, работать с CSS для управления общим стилем определенной части. Тег div — это блочный элемент, который можно использовать в качестве контейнера для объединения других HTML-элементов. Теги div можно использовать как инструмент строгой организации, и если вы пометите теги div идентификатором или классом, они будут работать еще лучше.

CMS

Система управления контентом (CMS) — это программная система, которая находится между интерфейсом веб-сайта (веб-сервер) и серверной офисной системой или процессом (создание и редактирование контента). Создатели, редакторы и издатели контента используют системы управления контентом для отправки, изменения, утверждения и публикации контента. «Содержимое» может включать в себя документы, таблицы, изображения, данные из баз данных или даже видео, которые вы хотите опубликовать на сайтах Интернета, интрасети или экстрасети.
